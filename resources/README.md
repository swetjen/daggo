# DAGGO Resources

Optional integration helpers for common external services:

- `resources/multidb` -> singleton DSN-based SQL connection helper
- `resources/s3resource` -> S3-compatible upload/download helpers
- `resources/ollama` -> Ollama text/image query client with retries
- `resources/playwright` -> remote Playwright scrape helpers

Each resource folder includes `*_todo.md` covering unresolved contract details and follow-up decisions.
